# GraCe (ICML 2024)

Gradient compressed sensing: A query-efficient gradient estimator for high-dimensional zeroth-order optimization

## Dependencies

The code was tested under the following dependencies:

- Python 3.10.12
- Tqdm 4.66.1
- NumPy 1.24.3
- NetworkX 3.1
- Matplotlib 3.7.4
- Seaborn 0.12.2
